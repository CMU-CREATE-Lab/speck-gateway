The Speck Gateway is a free, open-source application which
downloads data samples collected by the Speck to your computer.

You must have Java SE 6 or newer installed in order to run the software.

If you need help, please see the help documentation.

       

If your operating system is not one of the above, or if you have custom
configuration needs, please download the Zip file version of the software.
