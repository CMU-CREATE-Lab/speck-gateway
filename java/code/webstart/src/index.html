Speck

Speck is a low-cost particulate monitor currently under development by the CMU CREATE Lab as a tool for citizen science and personal exposure tracking. Through community-generated advocacy maps and data visualization, this device will enable communities and individuals to better understand and quantify the quality of the air they breathe.

The Speck Gateway is a free, open-source application which downloads data samples collected by the Speck to your computer and optionally allows you to upload the data to a BodyTrack Datastore server such as Fluxtream or Node Datastore Server. To download and run the Speck Gateway via Java Web Start, simply click the Launch button below.

version @version.number@

You will need Java SE 6 or later installed in order to run the software. If the software does not launch automatically, you will need to double-click the downloaded .jnlp file to launch.

Can't run the Speck Gateway via Java Web Start? Try the downloadable zip instead.
